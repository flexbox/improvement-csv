# improvement-csv

### Getting started

Launch on the terminal
```
node index.js
```
Export the results
```
node index.js > results.json
node index.js | gron | gron --ungron > result.json
```

### Dependencies

- [fast-csv](https://www.npmjs.com/package/fast-csv) Parse the file
- [lodash](https://www.npmjs.com/package/lodash)
